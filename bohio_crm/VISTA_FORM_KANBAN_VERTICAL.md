## 🚀 USO DE LA VISTA

### DESDE VISTA KANBAN:
```
1. Hacer clic en cualquier card del Pipeline Canvas
2. Se abre automáticamente esta vista form
3. Muestra toda la información en formato vertical
```

### NAVEGACIÓN:
```
• Ver todas las secciones scrolleando verticalmente
• Editar campos directamente
• Guardar con Ctrl+S o botón Guardar
• Cerrar con ESC o botón Cerrar
```

### BOTONES DE ACCIÓN:
```
📧 Enviar por Email:
   → Genera PDF del reporte
   → Envía al email del cliente (email_from)

🖨️ Imprimir:
   → Abre vista de impresión optimizada
   → CSS especial para print

✅ Seleccionar Esta:
   → Marca esta oportunidad como seleccionada
   → Útil para comparar múltiples opciones
```

## 📱 RESPONSIVE

### ESCRITORIO (> 1200px)
```
• Métricas: 4 columnas en grid
• Grupos: 2 columnas lado a lado
• Campos: Width completo
```

### TABLET (768px - 1200px)
```
• Métricas: 2 columnas en grid
• Grupos: 2 columnas más estrechas
• Padding reducido
```

### MÓVIL (< 768px)
```
• Métricas: 1 columna (vertical)
• Grupos: 1 columna
• Padding mínimo
• Fuentes más pequeñas
```
